## [1.0.0] - 2026-04-05

### Added
- Initial release of httix-http
- Ultra-lightweight axios replacement built on native Fetch
- Interceptors, retry, SSE streaming, rate limiting
- 5kB bundle, zero dependencies
- TypeScript-first with full type inference
- Dual ESM/CJS build
